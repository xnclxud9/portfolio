캐논아카데미 홈페이지 리뉴얼 프로젝트

2023.04.27 ~ 2023.05.19팀 프로젝트반응형
프로젝트 개요
브랜드를 선정하여 기존 홈페이지를 트렌드에 맞게 반응형 웹으로 리뉴얼
프로젝트 목적
반응형 웹 제작 능력 함양
팀원들과의 협업 경험으로 커뮤니케이션 능력 향상
javascript를 사용한 슬라이드 활용
디바이스별 테스트, 웹 표준 검사, 크로스 브라우징, 접근성 검사 경험
사용 언어
HTML5, CSS3, javascript, php
역할·업무
전체적인 디자인
메인 페이지의 멤버십 - 마우스 오버 시 전체 이미지 변경 기능 구현
메인 페이지의 이벤트 - 자동 슬라이드 기능 구현
로그인 페이지 배너 - 자동 슬라이드 기능 구현
전체적인 반응형 작업
다른 팀원이 작성한 코드 취합 및 오류 수정
느낀 점
처음으로 기획부터 개발까지 모든 프로세스를 경험하며 협업에 대해 배움
팀원이 작성한 코드를 수정하는 작업을 하며 디버깅 능력을 키움
프로젝트 자료 웹사이트 링크
